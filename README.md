# Проверка питания по QR

Откройте файл `index.html` в браузере или разместите его на любом хостинге.

## Как это работает
1. Человек показывает QR.
2. Вы сканируете QR камерой телефона.
3. Телефон открывает ссылку вида:
   `https://ВАШ-САЙТ/index.html?code=ABC123`
4. Если `code` есть в ссылке, страница показывает только ФИО и фото (без поля поиска).

## Требования к Google Sheets
В таблице должны быть колонки (названия можно на русском/английском):
- код (`QR`, `код`, `id`, `uid`, `бейдж`, `token`)
- ФИО (`ФИО`) или по частям (`ФИО 1`, `ФИО 2`, `ФИО 3`)
- фото (`фото`, `photo`, `image`, `avatar`, `photoUrl`)

Текущий Spreadsheet ID уже прописан в `index.html`:
`1xJr1arKCk1x_oiNT6dpt555zskHAbMEMYF3o3Vhy-qs`

## Важно
- В Google Sheets включите доступ: `Все, у кого есть ссылка -> Читатель`.
- В колонке фото указывайте прямую ссылку на картинку.
- Ссылки Google Drive вида `https://drive.google.com/open?id=...` тоже поддерживаются.
- В каждом QR должен быть URL вашего сайта с параметром `code`.

Пример для генерации QR:
`https://ВАШ-САЙТ/index.html?code=ABC123`
